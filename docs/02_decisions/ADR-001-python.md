# ADR-001 — Lenguaje objetivo Python

## Estado
Aceptada

## Contexto
El TFG necesita un alcance realista, defendible y con herramientas maduras para análisis de seguridad.

## Decisión
El lenguaje objetivo del MVP será Python.

## Justificación
- disponibilidad de herramientas como Bandit y Semgrep
- facilidad de integración con FastAPI
- menor complejidad que un enfoque multilenguaje
- mayor viabilidad para un TFG

## Consecuencias
- el MVP no será multilenguaje
- la evaluación se centrará en proyectos Python
