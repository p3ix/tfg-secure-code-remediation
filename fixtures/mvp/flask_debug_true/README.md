# Flask debug=True

## Objetivo
Caso simple de ejecución de aplicación Flask con modo debug activado.

## Caso incluido
- `app.run(debug=True)`

## Expectativa
La muestra debería ser detectable de forma sencilla y ser candidata a remediación asistida.

## Papel en el MVP
Caso simple y defendible para el flujo detect–repair–verify.
